📌 Overview
A cinematic, highly interactive personal portfolio built to showcase my skills, projects, and experience as a Full-Stack Engineer and Creative Developer. The site features immersive motion design, smooth animations, and a stunning visual experience — merging technical excellence with bold aesthetics.

Deployed Link:- https://maazportfolio1.vercel.app

✨ Features

🎬 Cinematic intro with video background and ambient sound toggle
🌀 Smooth scroll animations powered by GSAP
📱 Fully responsive design across all devices
🗂️ Sections: Home · About · Services · Projects · Certificates · Contact
📜 Downloadable Resume directly from the site
🏆 Certificate gallery with interactive card flip/expand
📬 Contact form for direct inquiries
⚡ Optimized performance via Next.js image optimization and code splitting
